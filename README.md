# Projeto CRUD Spring Boot - ADS 3º Período

Este é um sistema de gestão de funcionários e departamentos feito para a faculdade.

### 🛠️ Tecnologias
* **Java 17** com **Spring Boot**
* **PostgreSQL** (Banco de Dados)
* **JPA / Hibernate** (Para criar as tabelas sozinho)

### 📊 O que o projeto faz?
* Relaciona **Departamento** com **Funcionário** (Um para Muitos).
* Cria as tabelas automaticamente no pgAdmin.
* Permite listar e salvar dados através de rotas no navegador.
